# TareToUE5

> 专为 Unreal Engine 5 开发者打造的高效构建与调试工具

## 简介

TareToUE5 是一款功能强大的 VS Code 扩展，旨在简化 Unreal Engine 5 项目的开发流程。通过直观的可视化界面，开发者可以轻松管理项目构建、编译、调试和执行，无需在命令行和 IDE 之间频繁切换。

## 核心功能

### 📋 项目配置
- **可视化配置界面**：通过侧边栏面板轻松配置项目参数
- **路径管理**：支持 UE5 编辑器路径和项目路径的快速设置
- **配置验证**：实时验证配置有效性，避免因路径错误导致的构建失败
- **持久化存储**：自动保存配置，下次打开项目无需重复设置

### 🔨 构建管理
- **一键清理**：快速清理 Intermediate、Binaries 等中间文件
- **生成解决方案**：自动生成 Visual Studio 解决方案文件
- **重新生成**：清理并重新生成完整解决方案
- **实时进度**：构建过程中实时显示进度条和状态信息
- **详细日志**：输出面板显示完整的构建日志，便于问题排查

### 🚀 编译执行
- **编译项目**：使用 MSBuild 系统进行高效编译
- **多种配置**：支持 Debug、Development、Shipping、Test 等编译配置
- **启动项目**：一键启动 UE5 编辑器，无需手动查找可执行文件
- **取消操作**：支持随时取消正在进行的构建或编译任务

### 🐛 调试支持
- **调试执行**：编译并启动项目进行调试
- **执行不调试**：快速启动项目进行测试
- **状态监控**：实时显示调试状态和进度

## 技术特性

### 🎯 MSBuild 集成
- 完全模拟 Visual Studio 2022 的构建流程
- 通过命令行调用 MSBuild，无需打开 IDE

### ⚡ 高性能

### 🛡️ 稳定性
- 完善的错误处理机制
- 详细的错误提示和日志输出
- 支持进程取消和清理

## 安装使用

### 安装
1. 打开 VS Code 扩展市场
2. 搜索 "TareToUE5"
3. 点击安装

### 配置
1. 点击侧边栏的 "UE Builder" 图标
2. 在配置面板中设置：
   - **UE5 编辑器路径**：例如 `C:\Program Files\Epic Games\UE_5.6\Engine\Binaries\Win64\UnrealEditor.exe`
   - **项目路径**：选择你的 `.uproject` 文件
   - **编译配置**：选择所需的编译模式（默认为 Development）

### 使用
- **清理**：点击"清理"按钮清理中间文件
- **生成解决方案**：点击"生成解决方案"生成项目文件
- **重新生成**：点击"重新生成"清理并重新生成
- **编译**：点击"编译"按钮编译项目
- **调试**：点击"调试"或"执行"按钮启动项目

## 适用场景

- **独立开发者**：简化 UE 项目管理流程，提高开发效率
- **小型团队**：统一开发环境，减少配置差异
- **快速原型**：快速构建和测试项目想法
- **持续集成**：可作为 CI/CD 流程的一部分

## 系统要求

- **VS Code**：版本 1.85.0 或更高
- **操作系统**：Windows 10/11
- **Unreal Engine**：版本 5.0 或更高
- **Visual Studio**：2019 或更高版本（用于 MSBuild）
